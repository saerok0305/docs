# React-Simple-Documentation

## MDX 포맷으로 작성하는 React 기반의 도큐멘테이션

> React, MDX, Customizing, Documentation, Blog

블로그를 한다거나 뭔가에 대해 문서화를 하는 것은 정말 어려운 것 같습니다.

처음 마음을 먹었을 때는, Github Pages로 호스팅하고 Jekyll을 이용할 계획이었지만,
**내가 쉽게 커스터마이징이 가능한 페이지를 만들어보고 싶다**는 생각이 들면서 부터 머리가 복잡해지기 시작했습니다.

나름 공부했던 React를 이용해서 만들어 볼 수 있을까?
React에서 markdown을 이용해서 작성할 수 있는 방법이 있지 않을까?
Github Pages를 통해서 호스팅이 가능하겠지?
광고는 어떻게 붙이는 거지?
내 블로그가 검색이 될까?

이런 저런 생각만 하면서 많은 시간이 지나고 나서야, 마음을 먹을 수 있었고,
허술하긴 하지만, 내 스스로 도큐멘테이션 도구?를 만들어서 문서화를 시작할 수 있게 되었습니다.

**첫번째 문서화는, 내가 만든 도큐멘테이션 템플릿?을 다른 사람도 쓸 수 있도록 정리를 해보자**이고, React를 몰라도 사용하는데 무리가 없이 작성해 보자 입니다.
물론 React를 아신다면 더 나은 도구로 커스터마이징 가능하겠죠.


제 조악한 코드도 여러 사람들 의견으로 개선이 될 수 있지 않을까라는 기대도 하고 있습니다.


React-Simple-Documentation은 제가 문서화 도구를 만들면서 지은 이름입니다... 


이후 부터는, 문서화 혹은 도큐멘테이션(Documentation) 이라는 말 대신에 블로그(Blog)라는 용어로 통일하겠습니다.
(문서화 라는 말이 조금... 어색헤서요.)
